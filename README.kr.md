# Cpplanet
 오픈소스 c++ 에버플래닛 서버입니다.
## 구현 기능
  - 로그인
  - 월드 리스트
## 사용 방법
  - Release에서 Cpplanet.exe를 다운로드 받는다 or 직접 빌드한다.
  - 에버플래닛 한국 클라이언트를 구한다.
  - 클라이언트 폴더에서 Profile폴더 생성후, 안에 kr폴더를 생성하고, test.xml파일을 넣는다.
  - test.xml의 내용
  - `<profile></profile>`
  - 클라이언트 바로가기를 만들어서 바로가기의 대상란에 -profile:test -loginserveraddr:127.0.0.1:3431을 넣는다
  - 서버 실행 & 바로가기로 클라이언트 실행
