# Cpplanet
 오픈소스 c++ 에버플래닛 서버입니다.
## 구현 기능
  - 로그인
  - 월드 리스트
## 사용 방법
  - 클라이언트를 구한 후, Profile/kr/test.xml 파일을 생성합니다, 클라이언트 실행할때 인자를 넣어줍니다.
  - `GameClient.exe -profile:test -loginserveraddr:127.0.0.1:3431`
  - test.xml내의 내용은 상관 없습니다.
